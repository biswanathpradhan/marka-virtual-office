# Virtual Office - Laravel Application

A fresh Laravel installation with Virtual Office features.

## Installation

1. Install dependencies:
```bash
composer install
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
php artisan key:generate
```

3. Configure your database in `.env`

4. Run migrations:
```bash
php artisan migrate
```

5. Start the development server:
```bash
php artisan serve
```

## Virtual Office Features

The Virtual Office feature will be implemented on top of this fresh Laravel installation.
